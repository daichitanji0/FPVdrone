# FPVdrone
Minecraftに一人称で操作可能なドローンを追加するMOD(Fabric 1.21.11/Java)

## 📝概要
プレイヤーが操作可能なドローンを追加するModです。開発中。
ドローンの複雑な動きの計算や、マイクラ独自の仕組みを調べるために、AIを利用しています。
専門的な部分はAIの知恵を借りることで、一人では手が届かなかったクオリティに挑戦しました。

## ✨主な機能
- 操作感:WASDではなく、マウスを操縦桿のように使用する慣れが必要だけど動かしていて気持ちいい挙動。
- モデル:Blockbenchで制作したオリジナルドローンモデルの導入。
  <img width="1920" height="1128" alt="GeckoLib Entity 4 - Blockbench 2026_04_07 21_23_28" src="https://github.com/user-attachments/assets/2b374cca-440c-461c-8778-383bfdc3d829" />
- 通信途絶アニメーション:一定以上の速度でブロックに衝突したときにドローン死亡アニメーションを再生
